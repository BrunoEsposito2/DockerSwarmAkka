Public API: Base trait for all messages in the Akka cluster communication protocol.

Public API: Ping request message in the ping-pong communication protocol.

Initiates a ping-pong cycle by sending this message to a remote actor. The receiving actor should respond with a Pong message containing a reference back to the original sender.
